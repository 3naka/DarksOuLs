### MOD清单
##### LongerLisCombat：LI战斗延长（V0.2）
* 简单粗暴的提高Li们的持久力。

* 为PC贡献牛牛的肛交添加菜单选项，而原选项改为掌管PC被撅的情况。
* 注意：不会移除PC主动撅人的选项。
